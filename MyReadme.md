# Web Automation API using Playwright and Flask

This project provides a simple REST API built with **Flask** that triggers **browser automation using Playwright**.  
It can be used for automated testing, scraping, or browser-based workflows.

---

## 🚀 Features

- Browser automation using Playwright
- REST API using Flask
- Supports Chromium browser
- Easy to extend for additional automation tasks
- Works on Windows, Linux, and Mac
